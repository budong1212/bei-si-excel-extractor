# 贝思excel内容提取工具

> 支持多文件、多表格、百万行数据的 Excel 内容提取工具，带图形界面。

---

## ✨ 功能特性

- **多关键词搜索**：每行输入一个关键词，同时搜索多个内容
- **多种匹配方式**：包含 / 完全匹配 / 开头匹配，可选区分大小写
- **多文件支持**：同时处理多个 Excel 文件（`.xlsx` / `.xls` / `.xlsm`）
- **多表格支持**：自动处理一个 Excel 文件内的所有 Sheet
- **百万行优化**：使用 `read_only` 流式读取，内存占用低
- **拖拽支持**：可直接拖拽文件或文件夹到文件列表区域（需安装 `tkinterdnd2`）
- **进度 & 速度显示**：实时显示扫描进度、速度（行/秒）
- **智能表头合并**：相同表头的数据合并在一起；不同表头之间自动插入空行并重新添加表头

---

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install openpyxl tkinterdnd2
```

> `tkinterdnd2` 为可选依赖，不安装时拖拽功能不可用，其他功能正常。

### 2. 运行程序

```bash
python 贝思excel内容提取工具.py
```

---

## 📖 使用说明

1. **添加文件**：点击「添加文件」或「添加文件夹」按钮，或直接拖拽文件/文件夹到列表区域
2. **输入关键词**：在关键词框中每行输入一个搜索词
3. **选择匹配方式**：包含（默认）/ 完全匹配 / 开头匹配
4. **设置输出路径**：点击「浏览」选择结果保存位置
5. **开始提取**：点击「▶ 开始提取」，进度条实时更新
6. **完成后**：弹出提示，可选择直接打开结果文件

---

## 📋 输出格式

- 所有结果写入同一张 Sheet「提取结果」
- **相同表头的数据连续排列**
- **不同表头之间**：先一行空白，再写新表头（蓝色加粗），再写数据

## 🔧 系统要求

- Python 3.8+
- Windows / macOS / Linux（拖拽功能在 Windows 上测试最佳）
